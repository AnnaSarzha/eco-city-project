# Eco City Project

Платформа для мониторинга экологических проблем города.

## Описание
Проект позволяет пользователям сообщать об экологических проблемах в городе, 
отслеживать их статус и участвовать в улучшении городской среды.

## Технологии
- **Frontend:** React
- **Backend:** Node.js, Express
- **Database:** MySQL

## Структура проекта
- `/client` — React приложение (фронтенд)
- `/server` — Node.js сервер (бэкенд)
- `/database` — SQL скрипты базы данных
